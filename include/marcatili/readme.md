# Inventario de `include/marcatili/`

Esta pasta organiza os cabecalhos do namespace `marcatili`, separando as interfaces de entrada/saida, matematica e fisica.

## Subpastas

- `io/`: interfaces de parsing, serializacao e utilitarios de arquivo/texto.
- `math/`: interfaces do nucleo matematico compartilhado.
- `physics/`: interfaces das configuracoes, resultados e solvers fisicos.

## Arquivos

- `placeholder_app.hpp`: declaracao do fluxo placeholder usado por executaveis ainda nao implementados completamente.
- `readme.md`: este inventario da pasta `include/marcatili/`.
