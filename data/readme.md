# Inventario de `data/`

Esta pasta concentra os arquivos de entrada e os artefatos de saida do fluxo reprodutivel do projeto.

## Subpastas

- `input/`: casos de entrada usados pelos executaveis C++.
- `output/`: relatorios, tabelas numericas e figuras geradas a partir das execucoes.
