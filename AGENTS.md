# AGENTS.md

Objetivo:

Reproduzir o artigo de Marcatili (1969) sobre guia dielétrico retangular e acoplador direcional.

Regras:

1. Priorizar clareza, fidelidade e didática.
2. Não inventar fórmulas ausentes. pergunte que verifico e informo.
3. Marcar ambiguidades de OCR com TODO.
4. Separar física, matemática, I/O e executáveis.
5. Todas as saídas numéricas devem ir para CSV/JSON.
6. Todos os gráficos devem ser gerados por scripts Python externos.
7. Cada executável deve aceitar um arquivo de entrada.
8. Manter comentários explicativos no código.
