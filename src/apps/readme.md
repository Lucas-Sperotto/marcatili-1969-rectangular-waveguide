# Inventario de `src/apps/`

Esta pasta contem os executaveis do projeto. Cada arquivo e propositalmente fino: le a entrada, delega a fisica para `src/physics/` e usa `src/io/` para ler e escrever artefatos.

## Subpastas

Esta pasta nao possui subpastas.

## Arquivos

- `readme.md`: este inventario da pasta `src/apps/`.
- `reproduce_fig10.cpp`: ponto de entrada para o sweep numerico que reproduz a Fig. 10.
- `reproduce_fig11.cpp`: ponto de entrada para o sweep numerico que reproduz a Fig. 11.
- `reproduce_fig6.cpp`: ponto de entrada para a reproducao de um painel da Fig. 6.
- `reproduce_fig7.cpp`: ponto de entrada para a construcao do nomograma da Fig. 7.
- `reproduce_fig8.cpp`: ponto de entrada para a reproducao do caso metalizado da Fig. 8.
- `reproduce_table1.cpp`: ponto de entrada para a reproducao da Tabela I e da busca de cutoffs associados.
- `solve_coupler.cpp`: ponto de entrada para resolver um ponto isolado do acoplador direcional.
- `solve_single_guide.cpp`: ponto de entrada para resolver um caso isolado de guia unico.
