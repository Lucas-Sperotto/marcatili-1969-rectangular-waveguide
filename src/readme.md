# Inventario de `src/`

Esta pasta concentra as implementacoes C++ do projeto. Aqui ficam tanto os executaveis de entrada quanto os modulos reutilizaveis de I/O, matematica e fisica.

## Subpastas

- `apps/`: pontos de entrada executaveis do repositorio.
- `io/`: implementacoes da camada de parsing, serializacao e utilitarios de arquivo.
- `math/`: implementacoes do nucleo matematico compartilhado.
- `physics/`: implementacoes dos solvers fisicos e das rotinas de reproducao.

## Arquivos

- `placeholder_app.cpp`: implementacao do fluxo placeholder usado quando um executavel ainda nao possui modelo matematico final.
- `readme.md`: este inventario da pasta `src/`.
